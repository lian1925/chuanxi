Try【尝试】 React

作者： Lian

日期： 2018年05月23日 16:05

The easiest【最简单】 way to get started【开始】 with React is to use this Hello World example code on CodePen. You don’t need to install【安装】 anything; you can just open it in another tab and follow along as we go through examples【示例】. If you’d rather use a local development【开发者】 environment, check out the Installation section【章节】.

The smallest【最小】 React example【示例】 looks like this:

ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('root')
);

It renders【渲染】 a heading saying “Hello, world!” on the page.

The next few sections【章节】 will gradually【逐步】 introduce you to using【使用】 React. We will examine【测试】 the building blocks of React apps: elements【元素】 and components【组件】. Once you master【掌握】 them, you can create complex【复杂】 apps from small reusable【可复用】 pieces.

A Note【笔记】 on JavaScript

React is a JavaScript library【库】, and so we’ll assume【假想】 you have a basic【基本】 understanding of the JavaScript language【语言】. If you don’t feel very confident【自信】, we recommend【推荐】 refreshing your JavaScript knowledge so you can follow along more easily.

We also use some of the ES6 syntax【语法】 in the examples. We try to use it sparingly【保守】 because it’s still relatively【相对】 new, but we encourage【鼓励】 you to get familiar with【熟悉】 【箭头函数】arrow functions, classes, template literals, let, and const statements. You can use the Babel REPL to check【检查】 what ES6 code compiles【编译】 to.
