Package localization【本地化】 files【文件】

作者： Lian

日期： 2018年05月26日 12:05

Packages can contain【包含】 localized strings【字符串】 in JSON resource【资源】 files【文件】.

Translation【翻译】 resources【资源】

When building【构建】 the package, the pxt cli generates【生成】 various .json files with string resources【资源】 under the _locales folder【文件夹】、These are the original【原始】 string resource【资源】 files.

Adding【添加】 a translation【翻译】

Here are the steps【步骤】 for adding string【字符串】 translations to a package:

1、Translate【翻译】 the strings in these files and place them into target【目标】 language files【文件】 with the same【相同】 name.

2、Place the translated【翻译】 files under _locales/LANGISO/... where LANGISO is the language ISO code, like es for example【示例】、The language code is all lower【小写】 case.
3、Add the file reference【参考】 to the files section in pxt.json、Such as【比如】:

"files": [
    "README.md",
    "neopixel.ts",
    "_locales/ja/neopixel-strings.json",
    "_locales/zh/neopixel-strings.json"
],

See pxt-neopixel for an example of a localized【本地化】 package.
