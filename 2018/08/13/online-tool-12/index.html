nginx 安装 12 | 工具

作者： Lian

日期： 2018年08月13日 13:08

1 知识点

nginx 是一个高性能的web服务器，支持如下功能：
1、部署前端代码
2、代理服务器
3、负载均衡

2 实践

2.1 安装

2.1.1 centos安装

1、更新系统的软件库

yum -y update

2、安装epel-release

yum install epel-release

3、安装nginx

yum install nginx

2.2 常见命令

1、启动

systemctl start nginx

2、关闭

systemctl stop nginx

2.3 配置

配置文件位于：/etc/nginx/nginx.conf

1、将前端项目文件放置于 /data/home/dist目录下。

cd /data
mkdir home
mkdir dist

2、端口配置为 80

server {
  listen 80;
}

3、配置文件目录

server {
  listen 80;
  root /home/data/dist
}

4、综上，修改配置如下：

server {
    listen       80;
    root  /home/data/dist;

    charset utf-8;
    access_log  /var/log/fe.log;

    location /api/v1/  {
        proxy_pass http://demo.server.com/api/v1/;
    }
    location ^~ /index {
        try_files $uri /index.html;
    }
    location /business{
        try_files $uri /index.html;
    }

    location ^~ /basename/ {
        try_files $uri /index.html;
    }
}
