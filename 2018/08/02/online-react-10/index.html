Webpack DLL 优化 10

作者： Lian

日期： 2018年08月02日 10:08

1 知识点

wepack，在对项目代码进行打包时，需对两类代码进行打包，一类是自己写的代码，另一类是项目中引入的第三方库代码。

若是无差别地对这两类代码进行打包，工作量很大，将会极大影响打包效率。

因此，推荐的处理策略是：
1、第三方库代码更改率极低，故可提前将其编译，打包成dll文件，以供后期直接调用。
2、将自己的代码完全分离，每次只打包自身的代码。

这样，则极大降低了后期打包的工作量，提升webpack的编译打包效率。

1.1 修改文件 /build/dll.config.js

挑选第三方库，独立打包。

const webpack = require('webpack');
const path    = require('path');
const vendors = [
  'react',
  'antd',
  "react-dom",
  "react-loadable",
  "react-router",
  "react-router-dom",
  // ...其它库
];

module.exports = {
  entry: {
      "lib": vendors,
  },
  output: {
    path: path.resolve(__dirname),
    filename: '[name].[chunkhash].js',
    library: '[name]_[chunkhash]'
  },
  plugins: [
    new webpack.DllPlugin({
            path: path.resolve( 'manifest.json'),
            context: __dirname,
      name: '[name]_[chunkhash]'
    })
  ]
};

1.2 修改文件 /build/webpack.base.js


const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');
const CleanWebpackPlugin = require('clean-webpack-plugin');
const pkg = require('../package.json');
const webpack = require('webpack');

function resolve (dir) {
  return path.join(__dirname, '..', dir)
}

module.exports = {
  entry:{
    app: "./src/main.js",
    // vendor: Object.keys(pkg.dependencies)
  },
  output: {
    filename: '[name].[chunkhash].js',
    path: path.resolve(__dirname, '../dist')
  },
  resolve: {
    extensions: ['.js', '.jx', '.json'],
    alias: {
      '@': resolve('src'),
    }
  },
  mode: 'development',
  plugins: [
    new webpack.DllReferencePlugin({
      // context: __dirname,
      manifest: require('../manifest.json'),
  }),
    new CleanWebpackPlugin(['dist'],{root:path.resolve(__dirname,'..')}),
    new HtmlWebpackPlugin({
      filename: 'index.html',
      template: './src/index.html',
      inject: true
    })
  ],
  module: {
    rules: [
      {
        test: /\.(js|jsx)$/,
        exclude: /(node_modules|bower_components)/,
        use: {
          loader: 'babel-loader'
        }
      },
      {
        test: /\.css$/,
        // exclude: /(node_modules|bower_components)/,
        // include: [
        //     path.resolve(__dirname, 'node_modules')
        // ],
        use: [
          'style-loader',
          'css-loader'
        ]
      },
      {
        test: /\.scss$/,
        exclude: /(node_modules|bower_components)/,
        use: [
          "style-loader",
          "css-loader",
          "sass-loader"
        ]
      }
    ]
  }
};

1.3 修改文件 package.json

  "scripts": {
    "dll": "webpack --config build/dll.config.js",
    "build": "webpack --config build/webpack.prod.js",
    "dev": "webpack-dev-server --open --config build/webpack.dev.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  }

1.4 执行

1、编译打包第三方库

npm run dll

2、启动项目

npm run dev
