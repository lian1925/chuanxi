如何安装开发环境

1、什么nodejs？

答： nodejs提供了JavaScript运行的虚拟机环境，这是目前应用最广的开发环境。

我们的microbit项目，应用到了不少typescript代码，需要运行在nodejs环境。

2、如何安装 nodejs?

答：搜索nodejs官网，下载安装

3、什么是vscode?

答：常用的代码编辑器有: vscode（推荐），atom，sublime。

4、如何安装vscode?

答： 搜索vscode官网，下载安装

5、如何运行microbit?

答：
1、进入microbit项目目录

cd microbit的路径

2、安装项目依赖

npm install

3、安装pxt

npm i -g pxt

4、运行microbit

pxt serve

5、等待运行完成，自动打开浏览器。
