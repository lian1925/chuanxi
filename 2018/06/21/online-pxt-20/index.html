microbit 移植规划

1 如何获取代码

gitlab

1、什么是gitlab？
gitlab是免费的代码托管平台，与著名的GitHub功能相似，最大的优点在于，提供免费的私有代码托管服务。

2、如何注册gitlab?
gitlab提供了2种注册方式：1 使用GitHub账号关联注册（推荐） 2 独立注册

3、如何上传代码到gitlab?
答：
三步完成：
1 更新代码
git add .
2 添加更新日志，并确认
git commit -m “update sth”
3 提交云端
git push origin master

4、如何授权他人下载代码？
答：
1 让对方在自己的电脑生成 ssh公钥
``bash
ssh-keygen
cat ~/.ssh/id_rsa.pub

2 代码拥有者，将对方的ssh公钥加入gitlab白名单

5、如何下载代码？
答：
```bash
git clone git@gitlab.com:gudong/pxt.git

6、如何安装git?
答；
搜索git官网，下载安装

2 如何安装开发环境

1、什么nodejs？
答： nodejs提供了JavaScript运行的虚拟机环境，这是目前应用最广的开发环境。

我们的microbit项目，应用到了不少typescript代码，需要运行在nodejs环境。

2、如何安装 nodejs?
答：搜索nodejs官网，下载安装

3、什么是vscode?
答：常用的代码编辑器有: vscode（推荐），atom，sublime。

4、如何安装vscode?
答： 搜索vscode官网，下载安装

5、如何运行microbit?
答：
1、进入microbit项目目录

cd microbit的路径

2、安装项目依赖

npm install

3、安装pxt

npm i -g pxt

4、运行microbit

pxt serve

5、等待运行完成，自动打开浏览器。

3

3、microbit 运行与编译

1
安装webstorm，推荐使用的代码编辑器，理由是高效简洁，占用电脑资源少，开发不卡。
2
安装nodeJs，用于编译typescript（我们的代码有很大部分由typescript组成）。安装包已下载，见node-v8.9.3.pkg，直接双击安装。
3
安装pxt 命令行工具，开发时对项目代码进行处理。
安装方法，打开命令行工具，输入 npm install -g pxt ，敲打回车键，等待安装完成。

4
解压源码，见pxt-microbit源码.zip，建议删除中文字符，因为在某些环境下中文字符会导致代码编译错误。
打开命令行工具，进入源码文件夹，安装依赖包，在这里依赖包指项目引用的第三方函数库，若是安装不完整，将会影响项目的启动与运行。
安装依赖包的命令为： npm install

5
启动项目
打开命令行，输入 pxt serve ,等待项目启动完毕，打开浏览器，输入网址 http://localhost:3232/index.html#，即可观看效果
