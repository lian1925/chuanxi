3 知识点

3.1 DOCTYPE 声明

用途：告诉web浏览器，你将使用哪个版本的HTML标准来编写HTML代码。

3.2 用法

HTML 4 (传统)

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

HTML 5 (推荐)

<!DOCTYPE html>
