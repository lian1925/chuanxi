1 知识点

1.1 网页的构成

网页 = HTML(内容) + CSS(样式) + JavaScript(行为)

举例，示范如何查看淘宝的网页代码。

1.2 安装编辑器(vscode)

搜索vscode，进入官网，下载，完成安装。

1.3 HTML结构

html = head + body

1.4 HTML元素

html页面由多个Html元素组成。
我们的HTML元素有3个组成部分：开始标签，结束标签，内容。

1、开始标签：以<开头，以>结束，标签中的第一个单词，是标签的名称，描述它的功能。接下来的其他单词，是标签的属性，用于描述标签的行为。

2、结束标签：以</开头，以>结束。大多数标签是成对存在的，良好的编程习惯，是不能忘了写结束标签。如：省略

元素的结束标签，就是很多前端程序员容易犯的错误。

3、内容：包含在开始标签和结束标签中的内容，很多时候是文本。

如：

<p class="main">My cat </p>

2 实践

2.1 创建工程目录

  html1
  |- index.html

2.2 示例(Hello world)

修改文件 index.html

<head>
</head>

</body>
</html>
