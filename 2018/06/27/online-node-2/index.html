构建前端服务器 2 | Node.js

作者： Lian

日期： 2018年06月27日 13:06

1 知识点

1.1 前端服务器

现在越来越多的项目采用前后端分离的架构，此时，前端项目开发完成之后，需要独立部署。

此处，提供一种nodejs构建前端服务器，部署前端工程的方案。

基本思路如下： 前端项目编译打包=》生成静态文件 =》 nodejs服务器托管静态文件。

2 实践

2.1 修改工程目录结构

  node1
  |- package.json
+ |- app.js
+ |- /dist
+   |- index.html

2.2 修改文件 app.js

var express = require('express');
var app = express();

+ app.use(express.static('dist'));

app.get('/', function (req, res) {
  res.send('Hello NodeJs!');
});

var server = app.listen(3000, function () {
  var host = server.address().address;
  var port = server.address().port;

  console.log('Example app listening at http://%s:%s', host, port);
});

2.3 修改文件 /dist/index.html

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>Hello World</title>
    </head>
    <body>
        <h1>Hello World</h1>
    </body>
</html>
