初步 1 | Vue

作者： Lian

日期： 2018年06月27日 01:06

1 知识点

Vue的第一课，最简单的Hello world

1.1 初始化工程

包括步骤: 1创建目录 2初始化npm 3安装webpack和vue

mkdir vue1 && cd vue1
npm init -y
npm install webpack webpack-cli --save-dev
npm install vue --save

1.2 工程目录结构

  vue1
  |- package.json
+ |- index.html
+ |- /src
+   |- main.js
+ |- webpack.config.js

1.3 修改文件 index.html

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Demo</title>
</head>
<body>
  <div id="app">{{message}}</div>  <!-- Vue模板入口 -->
  <script src="dist/main.js"></script>
</body>
</html>

1.4 修改文件 /src/main.js

// import...from的语法是ES6的，需要用到babel，后面再说
// require的语法是Commonjs的，webpack已经实现了，可以直接使用
import Vue from 'vue/dist/vue.common.js';

new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue.js!'
  }
})

1.5 创建文件webpack.config.js

const path = require('path');

module.exports = {
  entry: './src/main.js',
  output: {
    filename: 'main.js',
    path: path.resolve(__dirname, 'dist')
  }
};

1.6 修改文件 package.json

{
  "name": "vue1",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
+    "dev": "webpack",

    "vue": "^2.5.16"
  }
}

1.7 运行，查看结果

1、打开终端

2、执行命令

npm run dev

3、查看结果(用浏览器打开index.html)
