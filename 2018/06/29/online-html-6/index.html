1 a 标签

a 标签，程序员喜欢称为锚(mao)，用于创建一个超链接。

1.1 a标签 语法：

a 标签 = <a + href= + 域名内容 + target= + _blank 或 _self + > + 标签内容 + </a>

1.2 a标签 属性

1、href 表示链接目标的URL。

2、target 规定在哪里显示链接的资源。
_self: 在当前HTML4或HTML5文档页面重新加载返回值。这个是默认值。
_blank: 以新的HTML4或HTML5文档窗口加载返回值。

如：

<a href="http://www.liangyl.com" target="_blank">小沙丘</a>

1.3 修改文件 index.html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>欢迎 登录</title>
    <meta name="description" content="梁英廉的个人博客，提供登录功能">
    <meta http-equiv="refresh" content="3">
  </head>
  <body>

    <!-- 标题内容 -->
    <h1>登录(Login)</h1>
    <h1>这是标题 1</h1>
    <h2>这是标题 2</h2>
    <h3>这是标题 3</h3>
    <h4>这是标题 4</h4>
    <h5>这是标题 5</h5>
    <h6>这是标题 6</h6>

    <!-- 段落内容 -->
    <p>
        天尊地卑，乾坤定矣。卑高以陈，贵贱位矣。动静有常，刚柔断矣。方以类聚，物以群分，吉凶生矣。　在天成象，在地成形，变化见矣。
    </p>

    <!-- 行内文本 -->
    <p>天尊地卑，<span>乾坤定矣</span>。卑高以陈，贵贱位矣。动静有常，刚柔断矣。方以类聚，物以群分，吉凶生矣。　在天成象，在地成形，变化见矣。</p>
  </body>

  <!-- a 标签 -->
  <a href="http://www.liangyl.com" target="_blank">小沙丘</a>
</html>

2 img 标签

img 标签，用于在页面显示图片。

2.1 img 属性

1、alt 规定了描述图像的替换文本。当图像无法显示时，代替图片显示在浏览器的内容。建议每个图像使用该属性，这样当图像无法正常显示，用户还是可以得到一些提示信息

2、width 图像的宽度

3、height 图像的高度

4、src 图像的URL，这是必填项，用于告诉浏览器在哪里可以找到图像

如：

<img src="mdn-logo-sm.png" alt="MDN" width="200px" height="200px">
