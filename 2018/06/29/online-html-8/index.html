1 form 标签

用途：创建表单，表单用于向服务器传输数据。

1.1 form 属性

1、action 定义提交表单的URL

2、method 定义发送form数据的HTTP方法

如：

<form action="form_action.asp" method="get">
  <p>First name: <input type="text" name="fname" /></p>
  <p>Last name: <input type="text" name="lname" /></p>
  <input type="submit" value="Submit" />
</form>

1.2 修改文本 index.html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>欢迎 登录</title>
    <meta name="description" content="梁英廉的个人博客，提供登录功能">
    <meta http-equiv="refresh" content="3">
  </head>
  <body>

    <!-- form -->
    <form action="form_action.asp" method="get">
      <p>First name: <input type="text" name="fname" /></p>
      <p>Last name: <input type="text" name="lname" /></p>
      <input type="submit" value="Submit" />
    </form>

</html>
