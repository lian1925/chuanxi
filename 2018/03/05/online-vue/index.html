是什么？

Vue.js，是一套用于构建用户界面的渐进式框架。

渲染

核心是允许用简洁的声明式语法，将数据渲染进DOM中。
如：
1、插入文本

<div id="app">
{{message}}
</div>

var app = new Vue({
  el:'#app',
  data:{
    message:'Hello Vue!'
  }
  })

2、绑定标签属性

<div id="app-2">
<span v-bind:title="message">
鼠标悬停
</span>
</div>

var app2 = new Vue({
  el:'#app-2',
  data:{
    message:new Date().toLocaleString()
  }
  })

3、条件

<div id="app-3">
<p v-if="seen">ok</p>
</div>

var app3=new Vue({
  el:'#app-3',
  data:{
    seen:true
  }
  })

4、循环

<div id="app">
<ol>
<li v-for="todo in todos">
{{ todo.text }}
</li>
</ol>
</div>

var app = new Vue({
  el:'#app',
  data:{
    todos:[
    {text: 'one'},
    {text: 'two'},
    {text: 'three'}
    ]
  }
  })

5、按键事件

<div id="app">
<p>{{message}}</p>
<button v-on:click="reverseMessage">逆转</button>
</div>

var app = new Vue({
  el:'#app',
  data:{
    message:'Hello Vue.js'
  },
  methods:{
    reverseMessage: function(){
      this.message = this.message.reverse()
    }
  }
  })

6、输入处理

<div id="app">
<p>{{message}}</p>
<input v-model="message">
</div>

var app = new Vue({
  el:'#app',
  data:{
    message:'Hello'
  }
  })

组件化

vue.js允许我们用独立和可复用的组件构建大型应用。这种编程方式，我们称为组件化。

组件
本质是一个拥有预定义选项的Vue实例。
1、注册

Vue.component('item',{
  template:'<li>代办项</li>'
  })

2、调用

<ol>
<item></item>
</ol>

好了，有人可能会说这样太呆板了。如果我们能从父作用域将数据传到子组件，看起来才酷炫。

Vue.component('item',{
  props:['todo'],
  template:'<li>{{ todo.text }}</li>'
  })

<div id="app">
<ol>
<item v-for="item in list" v-bind:todo="item" v-bind:key="item.id">
</item>
</ol>
</div>

var app = new Vue({
  el:'app',
  data:{
    list:[
    {id:0,text:'one'},
    {id:1,text:'two'},
    {id:2,text:'three'}
    ]
  }
  })

数据绑定
当vue实例被创建时，会向其data对象加入一些属性。当这些属性的值发生改变，视图将发生同步变化。

var data ={a:1}
var vm = new Vue({
  data:data
  })

vm.a === data.a

data.a = 2
vm.a //2

vm.a=3
data.a //3

注意，只有创建时，添加的属性才会双向绑定。
当然，如果你想阻止双向绑定，可以使用Object.freeze()方法

生命周期钩子
Vue实例在创建时都要经历一系列的初始化过程。如：
beforeCreate created beforeMount mounted beforeDestroy destroyed

new Vue({
  data:{a:1},
  created: function(){
    console.log(this.a)
  }
  })

语法

1、文本
数据绑定最常见的方式就是采用双括号的数据绑定

<p>{{msg}}</p>

2、原始html
双括号会将数据解释成普通文本，故如果想要输出HTML效果，需要使用v-html指令。

<p>{{rawHtml}}</p>
<p v-html="rawHtml"></p>

3、JavaScript语法支持
对于所有的数据绑定，Vue都提供了完全的JavaScript表达式支持。

{{number+1}}
{{message.reverse()}}
<div v-bind:id="'list-'+id"></div>

注意，只对表达式生效

//不生效
{{var a=1}}

指令
指令(directives)是带有v-前缀的特殊属性。其职责是：当表达式的值改变时，同步作用于DOM。

缩写
v-前缀作为一种视觉提示，用于识别vue的属性。不过，有些指令频繁用到，则会感到麻烦。因此vue为v-bind和v-on提供了缩写用法。

<a v-bind:href="url"></a>
<a :href="url"></a>

<a v-on:click="do"></a>
<a @click="do"></a>

计算属性
模板内的表达式非常便利，它的初衷是用于简单的计算。如果计算逻辑过大，则应该使用计算属性。

<div id="app">
<p>{{reverseMessage}}</p>
</div>

var vm = new Vue({
  el:'app',
  data:{
    message:'Hello'
  },
  computed:{
    reverseMessage: function(){
      return this.message.reverse()
    }
  }
  })

属性 vs 方法
上面的表达式可以使用方法调用，得到的结果

<p>{{reverseMessage()}}</p>

这两种方法有什么不同？
计算属性基于他们的依赖，只有相关依赖变化时才会重新求值。
相比之下，每次触发渲染，调用方法总会重新计算。

侦听属性
相对计算属性，Vue提供了一种更加通用的方法来监听vue实例的数据变化：watch，又称侦听属性。
示例

<div id="app">{{fullName}}</div>

var vm=new Vue({
  el:'#app',
  data:{
    firstName:'Li',
    lastName:'lian',
    fullName:'Li lian'
  },
  watch:{
    firstName: function(val){
      this.fullName=val+' '+this.lastName
    },
    lastName: function(val){
      this.fullName=this.firstName+' '+val
    }
  }
  })

若使用计算属性，则简单很多

var vm = new Vue({
  el:'#app',
  data:{
    firstName:'Li',
    lastName:'lian'
  },
  computed:{
    fullName: function(){
      return this.firstName+' '+this.lastName
    }
  }
  })

然而，在数据变化时，需要执行异步或开销较大时，侦听方法是最有用的。

<div id="watch">
<input v-model="question">
<p>{{answer}}</p>
</ div>

var vm=new Vue({
  el:'#watch',
  data:{
    question:'',
    answer:'until'
  },
  watch:{
    question: function(new,old){
      this.answer = 'waite',
      this.getAnswer()
    }
  },
  methods:{
    getAnswer: _.debounce(
      function(){
        if(this.question.indexOf('?')===-1){
          this.answer='?',
          return
        }
        this.answer='think'
        var vmm=this
        axios.get('https://')
        .then(function(response){
          vmm.answer=_.capitalize(response.data.answer)
          })
        .catch(function(error){
          vmm.answer='error'
          })
      },
      500
      )
  }
  })

这个例子中，watch 允许我们执行异步操作，限制操作频率，且在得到结果之前，设置中间状态，这是计算属性无法做到的。

样式
为HTML标签添加样式，是一个常见需求。我们应该如何操作呢？v-bind可以处理属性，而class正好是属性，因此可以使用v-bind。

<div class='static'
v-bind:class="{active:isActive}">
</div>

data:{
  isActive:true
}

渲染结果为：

<div class="static active"></div>

组件
当组件使用class属性，当新的class添加到其根元素上，取两者的并集。

Vue.component('com',{
  template:'<p class="foo bar"></p>'
  })

<com class="baz boo"></com>

渲染结果

条件

<h1 v-if="ok">yes</h1>
<h1 v-else>no</h1>

注意，v-else必须紧跟在v-if或v-else-if后，否则不被识别。

分组
v-if是一个指令，所以必须添加到一个元素。如果想对多个元素进行条件判断，则可以使用

<template v-if="ok">
<h1>a</h1>
<p></p>
</template>

复用
基于高效原则，vue通常会复用已有元素而不是从头开始。
有时这不符合实际需求，Vue提供一种方式来表达“这两种元素是完全独立的，不用复用它们”。只需要添加唯一值的key.

<template v-if="loginType === 'username'">
  <label>Username</label>
  <input placeholder="Enter your username" key="username-input">
</template>
<template v-else>
  <label>Email</label>
  <input placeholder="Enter your email address" key="email-input">
</template>

列表
使用v-for指令，可以将一个数组，渲染成一组列表元素。
其中需要使用item in items形式的特殊语法，items代表原数据数组，而item是数组的元素项。

<ul id="app">
<li v-for="item in items">
{{item.message}}
</li>
</ul>

var vm=new Vue({
  el:'#app',
  data:{
    items:[
    {message:'one'},
    {message:'two'}
    ]
  }
  })

v-for还支持一个可选的第二个参数作为当前项的索引。

<li v-for="(item,index) in items">
{{item}}-{{index}}
</li>

对象
一个对象往往有多个属性，可以通过v-for来遍历对象的属性。
分为三个层次：
1、只遍历属性

<ul id="app">
<li v-for="value in object">
{{value}}
</li>
</ul>

new Vue({
  el:'#app',
  data:{
    object:{
      name:'li',
      age:25
    }
  }
  })

2、遍历属性和键名

<li v-for="(value,key) in object">
{{key}}:{{value}}
</li>

3、遍历属性，键名和索引

<li v-for="(value,key,index) in object">
{{key}}:{{value}}:{{index}}
</li>

策略
默认情况下，为保证高效，v-for在更新已渲染过的列表时，采用”就地复用”策略，即仅是简单复用此处的每个元素，即使数据项顺序改变，它也不会重新匹配。

正式开发中，则建议提供vue一个提示，方便其跟踪每个节点的身份，实现重用和重新排序现有元素。具体的做法是为每项元素提供一个唯一的key属性。

<li v-for="item in items":key="item.id">
{{item}}
</li>

数组更新检测
数组有两种更新方法：变异与非变异
变异指直接在原有数组更新，相关方法有：
push pop shift unshift splice sort reverse

vm.items.push({message:'four'})

非变异指不改变原数组，返回一个新数组，用于替代原数组，相关方法有：
concat() slice() filter()

vm.items = vm.items.filter(function(item){
  return itme.message.match(/four/)
  })

注意事项
由于JavaScript的限制，Vue不能检测以下变动的数组。
1、当你直接利用数组的索引赋值，如：
vm.items[indexOfItem]= newValue
2、当你修改数组的长度，如：
vm.items.length = newLength
解决方案：
1、使用Array.prototype.splice
vm.items.splice(indexOfItem,1,newValue)
vm.items.splice(newLength)

对象更新检测
还是因为JavaScript的限制，Vue不能检测对象属性的添加与删除。

var vm = new Vue({
  data:{
    a:1
  }
  })
// invalid
vm.b=2

解决方案：
使用Object.assign()

var vm = new Vue({
  data:{
    userProfile:{
      name:'li'
    }
  }
  })
vm.userProfile = Object.assign({},vm.userProfile,{
  age:27
  })

过滤结果
有时，我们想要显示一个数组的过滤结果，但是不希望改变原数组，因为原数组可能还有其他用途。
策略，使用method方法

<li v-for="n in even(numbers)">
{{n}}
</li>

data:{
  numbers:[1,2,3,4]
},
method:{
  even:function(numbers){
    return numbers.filter(function(number){
      return number % 2 === 0
      })
  }
}

多元素
有时，我们希望处理多元素的显示。类似于v-if，这时可以使用

<ul>
<template v-for="item in items">
<li>{{items.msg}}</li>
<h1></h1>
</ul>

v-for 与 v-if 优先级
在同一节点时，v-for的优先级比v-if更高

<li v-for="todo in todos" v-if="!todo.isComplete">
{{todo}}
</li>

以上代码只显示了未完成的todos

组件
在自定义组件中，你可以像普通元素一样使用v-for

<com v-for="item in items":key="item.id"></com>

如果需要将数据传到组件内，我们要用props

</li>
</ul>

事件处理
我们可以用v-on监听DOM事件，并在触发时执行一些JavaScript代码。

<div id="app">
<button v-on:click="counter++">add one </button>
<p>{{counter}}</p>
</div>

var vm = new Vue({
  el:'#app',
  data:{
    counter:0
  }
  })

很多事件处理逻辑很复杂，所以直接将JavaScript代码写在指令中是不可行的。

<div id="app">
<button v-on:click="greet">Greet
</button>
</div>

var vm = new Vue({
  el:'#app',
  data:{
    name:'vue'
  },
  methods:{
    greet: function(event){
      alert(this.name)
    }
    if(even){
      alert(event.target.tagName)
    }
  }
  })

vm.greet() //=>'vue'

事件修饰符
修饰符由点开头，加指令后缀表示。作为对事件的修饰。

按键修饰符
在一些监听任务时，我们需要监听特定的按键事件。
我们的键盘，每个按键都对应一个唯一的keycode,故我们可以根据keycode监听事件。

<input v-on:keyup.13="submit">

记住所有的keycode是很困难的，Vue为常用的按键提供了别名,如：
.enter .tab .delete .space .up .down .left .right

<input v-on:keyup.enter="submit">

1、单行文本

<table>

组合
组件设计的初衷是为了组合使用，最常见的方式是父子组合，即组件A使用了组件B。它们之间必然要相互通信：父组件通过prop给子组件下发数据，子组件通过事件给父组件发送消息。
总结为：prop向下传递，事件向上传递。

Vue.component('child',{
  props:['message'],
  template:'<p>{{message}}</p>'
  })

<child message='hello'></child>

注意
因为HTML不分大小写，所以在HTML中需要转换为短横线命名。

Vue.component('child',{
  props:['myMessage'],
  template:'{{myMessage}}'
  })

<child my-message="hello"></child>
