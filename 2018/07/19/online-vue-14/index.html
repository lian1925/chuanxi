巧删console.log 14 | Vue

作者： Lian

日期： 2018年07月19日 16:07

1 知识点

console.log()是前端开发经常使用到的语句，用于打印程序中运行中遇到的数据。

这对于我们跟踪程序的执行状态非常有帮助。

然而，当程序开发完成，进入生产(production)阶段时，我们需要及时删除打印信息，这是因为打印输出可能会泄露程序的敏感信息。

本文目的在于介绍一种非常有用的删除打印信息的技巧，这里将会使用到一种babel插件。

1.1 安装依赖

npm install babel-plugin-transform-remove-console --dev-save

1.2 修改文件 .babelrc

{
  "env": {
    "production": {
      "plugins": ["transform-remove-console"]
    }
  }
}

注：此处的production限定只在生产模式下，删除console.log

1.3 done

配置完成，当你再次编译打包，将会发现所有的console.log信息将会不见。
