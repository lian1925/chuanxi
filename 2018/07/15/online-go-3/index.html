常量与变量 3 | Go

作者： Lian

日期： 2018年07月15日 16:07

1 知识点

常量，用于存储编程中的定值，又称不可改变量。

变量，用于存储编程中可以经常更改的值。

1.1 工程目录

  demo3
  |-  index.go

1.2 修改文件 index.go

package main

import "fmt"

func main() {
  // 常量
  const Pi = 3.14;
  // 此处若是更改Pi的值，将会报错
  // Pi = 2; 
  fmt.Println(Pi)

  // 变量
  var a = 1;
  a = 2

  fmt.Println(a)
}

1.3 运行，查看结果

1、编译，同一目录下生成执行代码index

go build index.go

2、运行

./index
