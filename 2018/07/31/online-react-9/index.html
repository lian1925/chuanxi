安装Ant Design 组件库 9

作者： Lian

日期： 2018年07月31日 13:07

1 知识点

Ant Design是蚂蚁金服推出，适用于企业级后台的React组件库。

1.1 安装

npm install antd --save

1.2 基础用法

基础用法，是最简单的写法，适合初学者。该用法是一次性引入完整的antd组件。

import { DatePicker } from 'antd';
ReactDOM.render(<DatePicker />,mountNode);

引入样式：

import 'antd/dist/antd.css';

1.3 进阶用法

进阶用法，指按照用户的需求，引入所需要用到的组件，用量刚刚好，不多不少，达至最佳的性能。

实现该目标，有两种可选方案：
1、手动引入
用户只需要手动引入所需的组件即可。举例，使用日期选择器组件DatePicker

import DatePicker from 'antd/lib/date-picker';  // 加载 JS
import 'antd/lib/date-picker/style/css';        // 加载 CSS

2、使用辅助库babel-plugin-import
该方案借助了辅助库，实现对js组件的按需引入，避免了每个组件都需手动处理的繁琐工夫。
步骤：
1、安装辅助库

npm install babel-plugin-import --save-dev

2、修改文件 .babelrc

{
  "plugins": [
    ["import", { "libraryName": "antd", "libraryDirectory": "es", "style": "css" }] // `style: true` 会加载 less 文件
  ]
}

3、引入所需模块，如：时间选取DatePicker

import {DatePicker} from 'antd';
