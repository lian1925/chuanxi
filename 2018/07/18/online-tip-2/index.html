常用命令 | Linux

作者： Lian

日期： 2018年07月18日 15:07

1 知识点

本节内容介绍常用的Linux经常使用的命令。

2 查看信息

1、系统版本

cat /proc/version

2、查看当前文件夹内文件清单

ls

3、查看cpu信息

lscpu

4、查看内存

free -m

5、查看磁盘

df -h

3 实用操作

3.1 关闭指定端口

web开发时，经常遇到端口占用问题，如：80端口被占用。此时，我们需要手动关闭指定端口，恢复使用。

1、查看占用端口的进程

lsof -i tcp:80

查看结果，如：

COMMAND   PID   USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
Google    563 huahua  110u  IPv4 0xd4de73d89a557025      0t0  TCP bogon:64435->14.116.179.205:http (CLOSED)

2、关闭该进程

kill -9 563

3.2 复制文件夹

命令格式

命令格式：cp [-adfilprsu] 源文件(source) 目标文件(destination)

参数说明：
-r 递归复制，适用于子目录的文件夹复制。
-u 若源文件比目标文件新，更新目标文件
-f 强制(force)执行

示例：将project/dist目录，复制到server/dist目录。

cp -rfu project/dist server
