服务器添加SSH特权 1

作者： Lian

日期： 2018年07月18日 15:07

1 知识点

我们用终端工具连接服务器时，需要输入ip 账户 密码，用于身份验证。

如果反复输入这些信息，会比较繁琐，那么是否有更简洁的方式呢？

答案是yes，通过为Linux服务器添加SSH特权。

2 本机

2.1 生成key

ssh-keygen

2.2 查看ssh公钥

cat ~/.ssh/id_rsa.pub

3 Linux 服务器

3.1 授权

1、命令行模式下，输入：

vim /etc/ssh/sshd_config

2、按 i 进入编辑状态

3、修改参数为：

PsswordAuthentiacation yes
PermitRootLogin yes

4、按 esc 退出编辑状态

5、输入 :wq ，按回车键，保存并退出。

3.2 将ssh公钥加入服务器白名单

1、创建白名单文件

vim ~/.ssh/authorized_keys

2、输入本地的ssh公钥

3.3 登录验证

1、对于Mac系统，步骤如下：

shell => 新建远程连接 => 输入ssh -p 22 root@14.116.179.161 => 点击连接
