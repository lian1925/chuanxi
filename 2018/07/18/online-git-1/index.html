使用技巧 1 | git

作者： Lian

日期： 2018年07月18日 11:07

1 知识点

本节内容，主要汇总git的实用操作场景。

1.1 推送新版本

1、添加更新内容

git add .

2、添加更新日志

git commit -m "something to tell."

3、推送至云版本

git push origin master

1.2 初始化git

为一个新的工程，实现初始化git。

1、进入工程根目录

cd /path to project

2、初始化命令

git init

3、设置 .gitignore
项目中有些文件，我们并不需要对其进行版本跟踪，此时可以指定git忽略文件。
如：第三方库文件(node_modelus文件夹)

.DS_Store
node_modules/
dist/

1.3 更改远端版本关联

1、删除旧的git源

git remote remove origin

2、添加新的git源

git remote add origin

3、推送，测试

git push origin master

1.4 拉取git代码

1、拉取全部代码

git fetch --all

2、指定代码版本

git reset --hard origin/master

1.5 代码合并

1、创建git分支，如：master -> develop

git branch develop

2、切换分支，如：master -> develop

git checkout develop

3、切换至主分支

git checkout master

4、融合develop分支

git merge develop

1.6 回滚github版本

1、查询历史版本

git log --pretty=oneline

2、恢复到指定版本

git reset --hard ...

3、推送远程服务器

git push -f -u origin master

4、更新

git pull
