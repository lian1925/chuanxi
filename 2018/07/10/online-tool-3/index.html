1 知识点

1.1 Node.js

Node.js是一个基于Chrome V8 引擎的JavaScript运行环境。

它使用了高效，轻量级的事件驱动，非阻塞I/O模型，以保证程序的高性能。

它的依赖生态系统，npm，是目前世界上最大的开源库生态系统。

1.2 安装

搜索nodejs，进入官网，下载，完成安装

1.3 检测node是否成功

1、打开终端

2、输入命令

node -v

3、如果正确输出nodejs的版本信息，则安装成功。否则安装失败。

1.4 检测npm是否成功

1、打开终端

2、输入命令

npm -v

3、如果正确输出npm的版本信息，则安装成功。否则安装失败。
