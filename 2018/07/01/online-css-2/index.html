选择器 2 | CSS

作者： Lian

日期： 2018年07月01日 11:07

1 知识点

1.1 选择器

想将样式应用于特定的HTML元素，需要想办法找到该元素。执行这一任务的规则称为css选择器。

在css中，有7大类选择器。

1.2 元素选择器

将HTML中的元素，直接作为选择器。
如：

body{
  color: green;
}

1.3 类选择器

为HTML元素命名，通过命名进行定位。

1、在HTML元素里，添加class属性，通常我们会配合div标签使用。

<div class="title">
login
</div>

2、定义css样式。

.class{
  color: red;
}

1.4 ID 选择器

为HTML元素命名，通过命名进行定位。

注意：与类选择器相比，一个HTML文档中可以有很多个同名的类选择器，却只能有一个同名的ID选择器

1、在HTML元素里，添加id属性，通常我们会配合div标签使用。

<div class="title">
login
</div>

2、定义css样式。

.class{
  color: red;
}

2 实践

2.1 创建工程目录结构

  css2
  |- index.html
  |- style.css

2.2 修改文件 index.html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>欢迎 登录</title>
    <link rel="stylesheet" type="text/css" href="./style.css" />
  </head>
  <body>

    <!-- 元素选择器 -->
    <h1>登录(Login)</h1>

    <!-- 类选择器 -->
    <div class = "border">你好。</div>
    <p class = "border">小沙丘</p>

    <!-- ID 选择器 -->
    <div id = "tip">我是ID</div>
  </body>
</html>

2.3 修改文件 index.css

h1 {
  color: red;
}

.border{
  border: 1px solid red;
}

#tip{
  border-bottom: 1px solid blue;
}
