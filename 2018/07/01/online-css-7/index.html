1 知识点

1.1 盒子模型 box model

盒子模型是css样式中，非常重要的概念。

盒子 = margin > border > padding > content

解释：
margin 外边距
border 边框
padding 内边距
content 内容
这里使用 > 代表包含关系，如：margin 包含border，border包含 padding。。。

1.2 盒子种类

1、content-box
仅仅将内容放于盒子内，而内边距，边框和外边距均放在盒子外面。
盒子大小 = 内容大小

2、border-box
内容，内边距，边框放在盒子内，而外边距放在盒子外面。
盒子大小 = 内容 + 内边距 + 边框

盒子模型种类的配置，一般放在body，保证对所有样式生效。
如：content-box border-box

body{
  box-sizing: border-box;
}
