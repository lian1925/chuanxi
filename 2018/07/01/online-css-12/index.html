动画 12

作者： Lian

日期： 2018年07月01日 16:07

1 知识点

常见的动画可以理解为：某种动画属性从一个值变化到另一个值。

动画属性包括：尺寸大小，数量，颜色

1.1 keyframses

用于定义动画的初始状态和结束状态。
如：

@keyframes move {
  from {
    background-color: red;
  }
  to {
    background-color: green;
  }
}
/* safari 和 chrome */
@-webkit-keyframes move {
  from {
    background-color: red;
  }
  to {
    background-color: green;
  }
}

1.2 animation

将定义好的动画与HTML元素绑定。
如：

div {
  animation: move 5s;
  -webkit-animation: move 5s;
}
