1 知识点

1.1 static 静态布局

默认布局
如：

body{
  position: static;
}

1.2 relative 相对布局

以原先位置为基点，进行相对偏移。一般情况下，相对布局充当绝对布局的容器块。
如：

body{
  position: relative;
}

1.3 abosute 绝对布局

以相对布局的父元素为基点，进行相对偏移。
如：

body{
  position: absolute;
}

1.4 fixed 固定布局

以浏览器窗口为基点，进行相对偏移
如：

body{
  position: fixed;
}
