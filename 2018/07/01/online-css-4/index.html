高级选择器 4 | CSS

作者： Lian

日期： 2018年07月01日 12:07

这是选择器的第三篇，将会介绍3种高级的选择器。

1 属性选择器

HTML元素通常会带有一些属性，我们可以根据属性及属性值，选择元素。

1、根据属性选择。

a[href]{
  color: green;
}

2、根据属性值选择

a[href="http://www.liangyl.com"]{
  color: blue;
}

2 伪类选择器

a 标签伪类，是典型的伪类选择器。

a:link{
  color: black;
}
a:visited{
  color:blue;
}
a:hover{
  color: red;
}
a:active{
  color: yellow;
}

3 伪元素选择器

常见的伪元素选择器有2种：1)before 2)after
1、before 用于在指定html元素前方插入新内容。

h1:before{
  cotent: 'lian'
}

2、after 用于在指定html元素末尾插入新内容。

h1:after{
  content: 'lian'
}
