1 知识点

1.1 小程序文件类型(4种)

1、JSON配置文件 .json后缀
2、WXML模板文件 .wxml后缀
3、WXSS样式文件 .wxss后缀
4、JS脚本逻辑文件 .js后缀

2 实践

2.1 创建新项目

1、菜单：项目 = 》新建项目 =》

2、填写项目参数：1）项目目录 2）AppID 3)项目名称 4）建立普通快速启动模板

3、生成工程项目

  demo1
  |- /pages
    |- /index
    |- /logs
  |- utils
    |- util.js
  |- app.js
  |- app.json
  |- app.wxss
  |- project.config.json

{
  "pages":[
    "pages/index/index", //描述index页面所在目录
    "pages/logs/logs" // 描述log页面所在目录
  ],
  "window":{
    "backgroundTextStyle":"light", 
    "navigationBarBackgroundColor": "#fff",
    "navigationBarTitleText": "WeChat",

  }
}
