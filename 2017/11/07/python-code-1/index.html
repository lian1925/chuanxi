课1 第一个程序 | Python

作者： Lian

日期： 2017年11月07日 09:11

前面学习完python环境的安装后，本课将会学习如何编写第一段python代码，并且让它运行起来。

本课会接触到第一个python 命令：print ，请先和它混个脸熟，学会拼写，记住他的名字。
在接下来的代码实践中，将会学习它的具体使用。

代码练习

打开atom，新建一个文件，敲打下面的代码，取名为lesson1.py。这个命名很重要，所有的python代码最好都以.py结尾。

print "Hello World!"
print "Hello Again"
print "I like typing this."
print "This is fun."
print 'Yay! Printing.'
print "I'd much rather you 'not'."
print 'I "said" do not touch this.'

重要的事情说三遍，说三遍。
1、每句代码一字一字手打。
2、每句代码一字一字手打。
3、每句代码一字一字手打。
千万别偷懒，复制代码。学习编程最重要的是自己敲代码。

OK，代码编写完，打开命令行终端，输入内容：
python lesson1.py
注意：请先用命令行进入lesson1.py所在的目录，若是不懂操作，尝试上网搜索

常见问题

1、如果你看到关于ASCII错误，尝试在python代码的最上面添加一行代码
# -*- coding: utf-8 -*-
这样你的脚本即可支持unicode 编码，不再出现字符乱码。
