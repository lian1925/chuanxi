课4 认识变量 | python

作者： Lian

日期： 2017年11月07日 11:11

在编程的世界里，变量是用来指代某个东西的名字。程序员的记性都不怎么好，变量名可以帮助他们更容易记住程序的内容。
如果他们在编写程序时忘记取好变量名，下次阅读之前的代码时他们将会很头痛。

1 知识点

这堂课，我们还将学习格式化字符串的技巧(format string)
%d 引用数字变量
%s 引用字符串变量
%r 引用调试(原始)变量

2 代码练习

my_name = 'Zed A. Shaw'
my_age = 35 # not a lie my_height = 74 # inches
my_weight = 180 # lbs
my_eyes = 'Blue'
my_teeth = 'White'
my_hair = 'Brown'
print "Let's talk about %s." % my_name
print "He's %d inches tall." % my_height
print "He's %d pounds heavy." % my_weight
print "Actually that's not too heavy."
print "He's got %s eyes and %s hair." % (my_eyes, my_hair)
print "His teeth are usually %s depending on the coffee." % my_teeth
# this line is tricky, try to get it exactly right
print "If I add %d, %d, and %d I get %d." % (my_age, my_height, my_weight, my_age + my_height + my_weight)

3 你应该看到的结果

$ python lesson4.py
Let's talk about Zed A. Shaw.
He's 74 inches tall.
He's 180 pounds heavy.
Actually that's not too heavy.
He's got Blue eyes and Brown hair.
His teeth are usually White depending on the coffee.
If I add 35, 74, and 180 I get 289.
$

4 加分项

1、试着修改变量的名字
2、尝试使用更多的格式化字符，如%s和%r

5 常见问题

1、%s %d 这些符号是什么意思？
在编程里，这叫格式化字符串，前半句格式好的字符串加%号，后半句放变量名。如同你去逛商店时，说我要买牛奶，鸡蛋，面包-》我要买？，？，？， (牛奶，鸡蛋，面包)。
这就是程序员的语法，I will buy %s, %s, %s,(milk,egg,bread)
